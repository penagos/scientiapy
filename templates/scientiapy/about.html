About Scientiapy

This is designed to be a modern, light-weight, open source answer exchange website. This site is written in Python ontop of the excellent Django Framework. It uses the Django ORM for its data model, as such, it can be easily retargeted to use other database backends.

Features

This is a non exhaustive list. Please refer to GitHub for the most up-to-date information.

  • Written in Python (may or may not be of value to you)
  • Clear abstraction of views and controller logic
  • Small data model
  • Responsive layout
  • Supports commenting on answers
  • Supports locking questions
  • Upvoting of questions and answers
  • User notification lists for questions (email users on new question or new answer posts)
  • Ability to tag questions
  • Full integration with the Django admin site
  • Very small and light-weight
  • Full Markdown support
  • Weekly digest emails
  • ... and its open source
